## Título
Login exibe mensagens de erro inconsistentes para credenciais inválidas

## Ambiente
- URL: https://opensource-demo.orangehrmlive.com
- Navegador: Chrome
- Sistema operacional: Windows

## Pré-condição
Usuário está na tela de login do sistema.

## Passos para reproduzir
1. Deixar o campo usuário vazio e preencher a senha.
2. Clicar em "Login".
3. Repetir o teste preenchendo usuário inválido e senha inválida.

## Resultado esperado
O sistema deveria exibir uma mensagem de erro padrão para qualquer tentativa de login inválida.

## Resultado atual
O sistema exibe mensagens diferentes dependendo da combinação de credenciais inválidas.
